1. Installation and Setup¶

1.1. Prerequisites¶

  • Python 3.5+ (may very well work with lower version of Python 3)
  • maplotlib
  • numpy

1.2. Installation¶

Import the module and call the “curlyBrace” function. See the “Example Code” section for more.

1.3. Built With¶

  • Visual Studio Code - The main development tool
  • Anaconda - The Python distribution used
  • Sphinx - Used to automatically generate documentations
  • Read the Docs - The theme for Sphinx
  • Windows 7 - The OS environment
